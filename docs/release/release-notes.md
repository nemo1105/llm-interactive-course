# 第一章页面发布说明

## 变更内容

- 首页改为整屏纵向课程价值展示，并提供进入第一章的入口。
- 第一章提供章节首页、普通对话演示、工具调用演示和下一章中性预留页。
- 演示页通过单一顶部工具栏驱动步进，左侧展示聊天状态，右侧展示标准时序图和 payload 浮层。

## 价值

- 复审者可以从一次最小 LLM 对话开始检查课程叙事、交互节奏和请求/响应解释是否清楚。
- 自动化已经覆盖主要路由、内容边界、步进同步、payload 可读性和短视口边界。

## 已知限制

- 当前版本使用 mock 数据，不接真实模型或真实天气 API。
- 当前版本尚未完成人工内容、视觉和演示节奏复审。
- fullPage.js 授权、licenseKey 或署名策略需要在正式分享前确认。
