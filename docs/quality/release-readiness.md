# 第一章页面发布就绪记录

## 当前判断

第一章页面已经通过本地自动化验证，可以进入人工内容、视觉和演示节奏复审；尚不是最终演讲版本。

## 验证证据

- 2026-06-24 运行 `pnpm run verify:iteration` 通过。
- 文档检查：`docs:lint` 通过。
- 类型检查：`typecheck` 通过。
- 单元测试：Vitest 2 个测试文件、13 条测试通过。
- 端到端测试：Playwright 11 条测试通过，覆盖章节首页、两个演示页、下一章预留页、步进同步、payload 浮层、JSON 树、API 格式切换、首页回归和短视口边界。

## 覆盖判断

| 覆盖类型 | 已有证据 | 缺口或待确认项 |
| --- | --- | --- |
| 成功路径 | 章节首页、两个演示、首页和下一章预留路由均由自动化覆盖。 | 人工复审中文表达、视觉密度和演示节奏。 |
| 失败路径 | 单元测试和 Playwright 覆盖禁用占位表达、未来消息提前出现、首页边界被破坏、payload 格式边界和中性预留页。 | 自动化不能判断演讲现场节奏是否自然。 |
| 边界或异常路径 | 覆盖短视口 payload、左右浮层定位、长工具调用时序、内部滚动、顶部工具栏稳定和 JSON 折叠展开。 | `@fullpage/react-fullpage` 当前运行时会输出 licenseKey 警告；发布前需确认购买授权或保留符合要求的 fullPage.js 署名方式。 |

## 发布阻断项

- 自动化验证失败。
- fullPage.js 授权、licenseKey 或署名策略未确认。
- 第一章页面仍出现禁用占位表达。
- 页面暗示模型自己执行工具。
- 固定章节路由或演示路由不可访问。
- 第一章事件无法查看实际传输数据。
- 演示不可步进，或左侧聊天状态和右侧时序图进度不同步。
- 标准时序图空白，或未来消息线、未来传输提前出现。

## 已接受风险

- 本次不接真实模型或天气 API，因此页面只验证教学交互，不验证真实服务集成。
- 视觉品质自动化只能覆盖结构和可见性，最终演示节奏仍需要人工复审。
